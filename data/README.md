# Data Preparation

This directory is used to store the raw and processed datasets required to reproduce the experiments in the submitted paper.

Due to file size and dataset license considerations, the raw datasets and generated processed files are not included in this repository. Users should download the datasets from their official sources and place them under the directory structure described below.

## Dataset Sources

The experiments use the following datasets:

- **SIFT1M**: http://corpus-texmex.irisa.fr/
- **CiaoDVD**: https://guoguibing.github.io/librec/datasets.html
- **KuaiRand-1K**: https://zenodo.org/records/10439422

Please follow the terms of use and license requirements of each dataset when downloading and using the data.

## Expected Directory Structure

Please organize the data directory as follows:

```text
data/
├── README.md
│
├── raw/
│   ├── sift1m/
│   ├── ciaodvd/
│   └── kuairand1k/
│
└── processed/
    ├── sift1m/
    ├── ciaodvd/
    └── kuairand1k_itemgraph_ctr/


The `raw/` directory stores the original downloaded datasets, while the `processed/` directory stores the files generated by preprocessing scripts and consumed by the C++ experiment programs.

## SIFT1M

Download SIFT1M from:

```text
http://corpus-texmex.irisa.fr/
```

Place the downloaded and extracted files under:

```text
data/raw/sift1m/
```

If your experiment program expects converted binary or text files, place the converted files under:

```text
data/processed/sift1m/
```

The SIFT1M experiments in the paper use the processed SIFT1M vectors together with a simulated social graph. Please make sure that the file names and paths match those specified in the corresponding C++ experiment entry file.

## CiaoDVD

Download CiaoDVD from:

```text
https://guoguibing.github.io/librec/datasets.html
```

Place the raw CiaoDVD files under:

```text
data/raw/ciaodvd/
```

Then run the preprocessing script:

```bash
python scripts/preprocess_ciaodvd.py
```

The processed files should be saved under:

```text
data/processed/ciaodvd/
```

Please make sure that the generated file names match the paths expected by the CiaoDVD C++ experiment program.

## KuaiRand-1K

Download KuaiRand-1K from:

```text
https://zenodo.org/records/10439422
```

Place the raw KuaiRand-1K files under:

```text
data/raw/kuairand1k/
```

Run the preprocessing script:

```bash
python scripts/prepare_kuairand1k_itemgraph_ctr.py \
  --kuairand_root data/raw/kuairand1k \
  --outdir data/processed/kuairand1k_itemgraph_ctr
```

This script trains BPR-MF embeddings, constructs item user-aggregate embeddings, builds an item co-click graph, fits calibration parameters, and exports the files required by the C++ KuaiRand-1K experiment.


The C++ KuaiRand-1K experiment reads  files from:

```text
data/processed/kuairand1k_itemgraph_ctr/
```

## Notes

1. The raw datasets are not included in this repository.
2. The processed files are generated locally by the preprocessing scripts.
3. Please keep the directory names consistent with the paths used in the C++ experiment programs.
4. If you use a different data directory, please update the corresponding path in the C++ experiment entry file or pass the path as a command-line argument.
